TinyMaker Web Slicer (in-browser)

Drop an STL/OBJ, set printer/profile, slice and download a ZIP that contains a top-level job folder with 1.png..N.png ready for the TinyMaker SD card.

Printer / Export Profile

Note: Firmware special rule — if device layer height > 0.06mm the TinyMaker firmware will load odd-indexed images (1,3,5...) so the exporter will automatically slice at half-step when needed.

Instructions

  1. Slice; download job.zip
  2. Extract job.zip to the SD card root so you have /job/1.png, /job/2.png,...
  3. On the TinyMaker set Layer_Height to your Device Layer Height chosen above (e.g., 0.05mm). If >0.06, firmware auto-skip applies.
